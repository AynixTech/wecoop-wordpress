
# WECOOP Institutional Website

Homepage istituzionale WECOOP sviluppata con componenti modulari in HTML, CSS e JavaScript (senza React).

## Struttura principale

- `index.html`: entrypoint della pagina
- `src/css/theme.css`: token tema WECOOP (colori, tipografia, spaziature)
- `src/css/components.css`: stili dei componenti UI
- `src/js/components.js`: componenti HTML generati via funzioni
- `src/js/main.js`: bootstrap pagina e interazioni (menu mobile, reveal, form)

## Avvio progetto

1. `npm install`
2. `npm run dev`

## Build produzione

1. `npm run build`
